# EduNexus Landing Page

A modern, professional landing page for EduNexus — an AI-powered learning platform for high schools.

## Sections

- **Navigation** — Sticky header with links to Features and Contact
- **Hero** — Platform overview and call-to-action
- **Features** — Six capability cards (AI tutoring, analytics, assessments, and more)
- **Contact** — Demo request form with email and phone details

## Run locally

Open `index.html` in your browser, or serve the folder with any static server:

```bash
# Python
python3 -m http.server 8080

# Node (if you have npx)
npx serve .
```

Then visit `http://localhost:8080`.
